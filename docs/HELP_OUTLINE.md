# Breeder Management App - Help Documentation Outline

## 1. Getting Started

### 1.1 Account Setup

- **Creating an Account**: Sign up with email and password
- **Logging In**: Access your breeder dashboard
- **Breeder Profile Setup**: Configure your kennel information, contact details, and branding

### 1.2 Initial Configuration

- **Email Settings**: Configure EmailJS for automated reminders and communications
- **Reminder Settings**: Set up pickup and deposit reminder preferences
- **Public Website Setup**: Enable/disable your public breeder website and litter pages

---

## 2. Dog Management

### 2.1 Adding Dogs

- **Add New Dog**: Enter basic information (name, breed, sex, date of birth)
- **Photos**: Upload and manage dog photos
- **Registration Details**: Track AKC/CKC registration numbers and status
- **Program Status**: Set as breeding dog, guardian home, external stud, or retired

### 2.2 Dog Profiles

- **Viewing Dog Details**: Access complete dog information
- **Health Tracking**: Record vaccinations, vet visits, deworming, and health events
- **DNA Profiles**: Store genetic testing results including coat color, traits, and health markers
- **Weight Tracking**: Monitor growth with weight entries and charts
- **Pedigree Tree**: View 4-generation family tree with interactive cards

### 2.3 Managing Your Dogs

- **Editing Dog Information**: Update details, photos, and registration status
- **Filtering Dogs**: View by program status (all, breeding, guardian, external stud, retired)
- **Marking Dogs**: Set dogs as deceased or retired with date tracking
- **Deleting Dogs**: Remove dog records (warning: affects litter and pedigree connections)

---

## 3. Heat Cycle Management

### 3.1 Recording Heat Cycles

- **Add Heat Cycle**: Track start date, end date, and breeding status
- **Breeding Information**: Record if bred and breeding dates
- **Notes**: Add observations about the heat cycle

### 3.2 Heat Predictions

- **Automatic Calculations**: System predicts next heat based on cycle history
- **Dashboard Overview**: View upcoming heats for all breeding females
- **Cycle Length Tracking**: Monitor average cycle lengths per dog

---

## 4. Litter Management

### 4.1 Creating Litters

- **Litter Setup**: Select sire and dam, set breeding date and expected/actual birth date
- **Litter Naming**: Create memorable litter names (e.g., "Summer 2024 Litter")
- **Status Tracking**: Monitor litter status (planned, born, weaned, ready)

### 4.2 Puppy Management

- **Add Puppies**: Record individual puppies with names, sex, colors, and markings
- **Weight Tracking**: Log puppy weights with visual charts
- **Health Records**: Track vaccinations, deworming, and vet visits per puppy
- **Microchip Information**: Store microchip numbers and registration details
- **Registration**: Track AKC/CKC puppy registration numbers

### 4.3 Care Schedules

- **Templates**: Create reusable care schedule templates
- **Daily Routines**: Set up recurring tasks (feeding, cleaning, socialization)
- **Milestones**: Track developmental milestones (eyes open, first steps, etc.)
- **Drag-and-Drop**: Organize tasks with intuitive interface

### 4.4 Puppy Sales

- **Status Management**: Mark puppies as available, reserved, sold, or kept
- **Pricing**: Set individual puppy prices
- **Buyer Assignment**: Link puppies to customer records
- **Deposit Tracking**: Record deposits and payment information
- **Contract Generation**: Create PDF contracts with digital signatures

## 22. Glossary

**Dam**: Female parent dog
**Sire**: Male parent dog  
**Heat Cycle**: Female dog's estrus/fertility period
**Guardian Home**: Program where dogs live with families but return for breeding
**External Stud**: Male dog not owned by breeder but used for breeding
**Whelping**: Process of giving birth to puppies
**Litter**: Group of puppies born at the same time
**Pedigree**: Family tree showing ancestors
**AKC**: American Kennel Club
**CKC**: Canadian Kennel Club (or Continental Kennel Club)
**Microchip**: Identification chip implanted in puppies
**Deposit**: Advance payment to reserve a puppy
**Co-ownership**: Shared ownership between breeder and buyer
**Breeding Rights**: Permission for buyer to breed purchased dog
